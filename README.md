# KazakhBike

Сотқа арыз жазуды автоматтандыратын WPF қосымша.

## Негізгі мүмкіндіктер:
- Қазақша және орысша тіл таңдау
- Сұрақ-жауап логикасы арқылы арыз құрастыру
- Дайын жауаптарды таңдау немесе өз нұсқаңды енгізу
- Алға/артқа навигация
- Қолданушы жауаптарына байланысты динамикалық сұрақтар

## Мақсаты:
Қолданушыларға ұзақ кезек күтпей, тез әрі ыңғайлы түрде онлайн арыз жазуға көмектесу

## Технологиялар:
- C#
- WPF
- Download and run the application (Windows .exe)
